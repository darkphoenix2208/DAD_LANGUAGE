A dictionary for converting standard programming concepts (C++/JS) to **Dad Lang**.

| Concept | C++ / JavaScript | Dad Lang Syntax | Meaning / Context |
| :--- | :--- | :--- | :--- |
| **Entry Point** | `main() { ... }` | `idhar aao` ... `jao padhai karo` | "Come here" ... "Go study" |
| **Print** | `cout <<` / `console.log` | `jawab de <msg>` | "Answer me!" |
| **Variable** | `let` / `var` | `ye sambhal <name> = <val>` | "Handle this" |
| **Constant** | `const` | `parampara <name> = <val>` | "Tradition" (Immutable) |
| **Loop** | `while (cond)` | `jab tak main na bolu (cond)` | "Until I say so" |
| **If** | `if (cond)` | `agar sharam hai (cond)` | "If you have any shame" |
| **Else If** | `else if (cond)` | `nahi to bhai (cond)` | "Otherwise brother" |
| **Else** | `else` | `warna belt` | "Otherwise... belt!" |
| **True** | `true` | `sharmaji_ka_beta` | "Sharma ji's son" (Always correct) |
| **False** | `false` | `nalayak` | "Useless/Incompetent" |
| **Null** | `null` | `khaali_dimaag` | "Empty brain" |
| **Break** | `break` | `bas kar bhai` | "Stop it brother" |
| **Continue** | `continue` | `agla dekh bhai` | "Look at the next one" |
| **Try** | `try` | `chori chuppe` | "Secretly" |

## Example Translation

### C++ Code
```cpp
int main() {
    int pocketMoney = 500;
    while (pocketMoney < 1000) {
        cout << "Paisa dedo";
        pocketMoney += 100;
        if (pocketMoney > 800) break;
    }
    return 0;
}
```

### Dad Lang Code
```javascript
idhar aao
    ye sambhal pocketMoney = 500;
    
    jab tak main na bolu (pocketMoney < 1000) {
        jawab de "Paisa dedo";
        pocketMoney += 100;
        
        agar sharam hai (pocketMoney > 800) {
             bas kar bhai;
        }
    }
jao padhai karo
```
